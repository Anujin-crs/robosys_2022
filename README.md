# plus command
Plus calculation program which add numbers read from standard input.

## Install package
```bash
$ git clone https://github.com/Anujin-crs/robosys_2022/ 
```

## Usage 
Example: 
```bash
$ seq 5 | ./plus  # to calculate the sum of integers from 1 to 5
```

## Required software
* Python
  * Test only: 3.7~3.10

## Test environment
* Ubuntu 22.04
